# PhysCode ⚛️
**Interactive Computational Physics & C++ Learning Platform**

PhysCode is a modern, interactive web application designed to help B.Sc. Semester 5 students learn Computational Physics and C++ programming. Built with a focus on visual learning, it transforms traditional syllabus topics into engaging, interactive experiences.

## 🚀 Live Demo
**[View the Live Website Here](https://AshishYadavStudio.github.io/PhysCode/)**

## ✨ Features
- **Premium Dark Mode UI**: A beautiful, distraction-free environment utilizing glassmorphism and modern design principles.
- **Interactive Visualizations**: Includes memory visualizers, expression evaluators, loop steppers, and dynamic code simulators.
- **C++ Execution Simulators**: See C++ code run step-by-step directly in your browser without needing a compiler.
- **Mathematical Rendering**: Complex physics and math formulas are beautifully rendered using KaTeX.
- **Client-Side Only**: Extremely fast and lightweight. No backend required—runs entirely in the browser using Vanilla JavaScript, HTML5, and CSS3.

## 📚 Syllabus Covered
The platform currently covers the B.Sc. Semester 5 Computational Physics syllabus:

### Unit I: Fundamentals & C++ Core
* History and Generations of Computers
* Hardware & Memory Architecture
* Introduction to System & Application Software
* Programming Languages (Low, Middle, High-level)
* C++ Introduction, Data Types, and Operators
* Variables, Scope, and I/O Statements (cin/cout, printf/scanf)

### Unit II: Control Flow, Arrays & Algorithms
* Control Statements (if/else, switch, ternary)
* Loop Constructs (while, do-while, for)
* Arrays (1D and 2D Matrices) & Structures
* Computational Physics Algorithms (Area/Volume, Matrices, Patterns)

## 🛠️ Technology Stack
- **HTML5 & CSS3**: For structure and responsive, premium styling.
- **Vanilla JavaScript**: For logic, routing, and interactive components.
- **[KaTeX](https://katex.org/)**: For fast math typesetting.
- **[Highlight.js](https://highlightjs.org/)**: For C++ syntax highlighting.

## 💻 Local Development
Since the project uses Vanilla JavaScript without any build tools, running it locally is incredibly simple:

1. Clone the repository:
   ```bash
   git clone https://github.com/AshishYadavStudio/PhysCode.git
   ```
2. Navigate into the directory:
   ```bash
   cd PhysCode
   ```
3. Start a local server (e.g., using Python):
   ```bash
   python -m http.server 8000
   ```
4. Open your browser and navigate to `http://localhost:8000`.
